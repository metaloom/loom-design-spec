# Model

A **Model** describes the structure of a **Content** by listing various fields.

## Properties

* Name
* Fields (see below)

## Field Types

* Text / List
* JSON / List
* Number / List
* Boolean / List
* Date / List
* Reference / List
* Model / List (For nesting)
* Asset / List

## Requirements

- Models can be nested
- Models are directly accessible (no need to assign them)
